# 星尾归航 / STARTAIL

原创三渲二风格互动叙事 H5 页游。当前版本包含使用 Three.js 实时渲染的第一章可玩关卡，临时低模用于验证第一人称猫咪移动、探索与互动玩法。

## 运行

直接双击 `index.html` 即可运行。为了获得更稳定的音频与缓存体验，也可以在本目录启动静态服务器：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 场景 01 动画预览

打开 `scene-01-preview.html` 可单独观看序章短片“失重乱流”。成片为 15 秒、1280×720、24 fps 的 H.264/AAC MP4，并已接入游戏开场。

互动节点包括：轻触项圈唤醒米粒、连续触碰漂浮光点、长按按钮或空格抵抗乱流。项圈信号低于安全线时可以重试，成功后自动进入“漂流货舱”。

源关键帧与成片位于 `assets/video/scene-01/`，可通过 `tools/render_scene01.py` 重新渲染镜头运动、粒子、警报闪烁与声音。

## 第一章操作

- W / A / S / D：移动
- 鼠标：控制视角
- Shift：奔跑
- 空格：跳跃
- E：互动；在电缆前需要长按
- Q：开启或关闭猫咪感知
- Esc：暂停
- 移动端横屏使用左侧摇杆、右侧拖动视角和动作按钮

## 第一章流程

- 探索失联舱段并拾取项圈
- 开启感知，沿猫爪足迹寻找损坏的电缆
- 躲避巡逻机的扫描并利用箱体遮挡
- 长按互动抓挠电缆，恢复维修舱门供电
- 穿过舱门并抵达第一章检查点

## 重新构建三维脚本

```powershell
pnpm install
pnpm run build:3d
```

生成式背景素材位于 `assets/images/`，第一章实时三维源代码位于 `chapter-one-3d.js`，浏览器实际加载打包后的 `chapter-one-3d.bundle.js`。
